Ethernet: Estándares y funcionamiento

Ethernet (IEEE 802.3)

El protocolo más utilizado en redes LAN cableadas, que ofrece alta velocidad y confiabilidad mediante cables de cobre o fibra óptica.

Wi-Fi (IEEE 802.11)

Protocolo estándar para redes LAN inalámbricas que permite la transmisión de datos sin cables, usando ondas de radio.

DNS (Domain Name System)

Protocolo utilizado para la resolución de nombres de dominio a direcciones IP, facilitando la conectividad en redes LAN e Internet.

Protocolo TCP/IP

TCP/IP es un conjunto de protocolos que facilitan la comunicación entre dispositivos en una red. TCP se ocupa de garantizar que los datos lleguen correctamente, mientras que IP se encarga de dirigir esos datos hacia el destino adecuado a través de una dirección única.

🕸️ Capa de Acceso a la Red

La capa de acceso a la red se encarga de la transmisión física de los datos a través del medio de comunicación, ya sea un cable (como en Ethernet) o señales inalámbricas (como en Wi-Fi).

Protocolos:

ARP

Wi-Fi

Ethernet

📶 Capa de Internet

La capa de Internet se encarga de dirigir los paquetes de datos de un dispositivo a otro a través de la red, utilizando el protocolo IP (Internet Protocol). Esta capa es responsable del direccionamiento y el enrutamiento de los paquetes de datos, garantizando que lleguen al destino correcto.

Direcciones IP:

IPv4

IPv6

Tipos de IP:

IP Privada

IP Pública

💻 Capa de Transporte

La capa de transporte asegura que los datos se entreguen de manera fiable entre los dispositivos de una red, garantizando que la comunicación de extremo a extremo sea adecuada.

Protocolos:

TCP

UDP

🖧 Capa de Aplicación

En esta capa residen los protocolos que interactúan directamente con las aplicaciones del usuario y proporcionan los servicios necesarios para la comunicación de datos entre los dispositivos.

Protocolos:

HTTP

FTP

DNS

SMTP
